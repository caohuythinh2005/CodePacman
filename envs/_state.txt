# GameState cho Pacman
#
# Lớp này lưu trữ trạng thái của trò chơi Pacman, bao gồm:
#
# 1. object_matrix: ma trận [H, W] biểu diễn tổng quan các ô trên bản đồ
#       - Chứa thông tin loại đối tượng tại mỗi ô (trống, tường, thức ăn, capsule, Pacman, ghost)
#       - Tham khảo các hằng số trong layout.py để biết giá trị cụ thể
#
# 2. pacman: struct AgentInfo
#       - x, y: tọa độ
#       - dir : hướng di chuyển (0=STOP, 1=NORTH, 2=EAST, 3=SOUTH, 4=WEST)
#
# 3. ghosts: danh sách GhostInfo, mỗi ghost gồm:
#       - x, y: tọa độ
#       - dir : hướng di chuyển
#       - scared_timer: thời gian còn lại trạng thái scared
#       - Số lượng ghost linh hoạt
#
# 4. score: điểm hiện tại của trò chơi
# 5. win / lose: trạng thái thắng / thua
#
# Lưu ý:
#   - Logic game nên dựa vào ghosts[i].scared_timer cho từng ghost
#   - Không còn infor_vector; mỗi agent xử lý trạng thái riêng và serialize nhanh
